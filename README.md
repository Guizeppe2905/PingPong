# PingPong
Настольный теннис игра с элементами Combine
***
Простая игра в настольный теннис, созданная для отработки элементов
***
### Stack
***
UIKit
SpriteKit
GameplayKit
Combine
UserDefaults
CAShapeLayer
Паблишеры комбайн используются в учебных целях для отработки их применения на практике.
